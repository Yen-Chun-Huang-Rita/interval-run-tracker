# 🏃 間歇跑訓練紀錄器 (Interval Run Logger)

這是一個使用 Python 撰寫的輕量級間歇跑計時與紀錄工具。它可以在終端機引導你完成高強度與低強度的間歇訓練，並自動將訓練成果記錄下來。

## 🌟 功能特色
* **自訂訓練菜單**：可自由設定「跑步秒數」、「休息秒數」與「循環組數」。
* **終端機動態倒數**：清晰顯示目前階段與剩餘時間，並帶有提示音（依系統支援而定）。
* **自動記錄進度**：訓練結束後，會自動生成並更新 `workout_log.csv` 檔案，方便後續使用 Excel 或 Google 試算表追蹤體能進步狀況。

## 🚀 如何使用
1. 確保電腦已安裝 Python。
2. 下載或 Clone 本專案。
3. 在終端機執行以下指令：
   ```bash
   python interval_run.py
