# 🏃‍♂️ 間歇運動計時器 (Interval Workout Timer)

這是一個專為「快走」與「慢跑」設計的輕量級網頁版間歇運動計時器。無需下載安裝任何 App，只需透過瀏覽器開啟即可使用，完美支援電腦與智慧型手機。

## ✨ 核心功能特色

* **⏱️ 高度客製化**：自由設定「快走」與「慢跑」的分鐘數（支援小數點，例如 0.5 分 = 30 秒）及總循環組數。
* **📊 預估總時間**：調整參數時，系統會即時自動計算並顯示本次訓練的總花費時間。
* **🔊 智慧音效提示**：每個階段最後 5 秒自動發出倒數短音，切換階段時發出長音。內建特殊機制，完美突破 iOS Safari 的音效限制。
* **📱 螢幕防休眠 (Wake Lock)**：訓練啟動後，自動防止手機螢幕進入休眠變黑，確保計時與音效不中斷。
* **⏸️ 暫停與繼續**：運動途中若需休息或喝水，可隨時暫停與恢復計時。
* **💾 完整的紀錄管理**：
  * 自動將訓練成果（日期、設定、總時間）儲存於瀏覽器本地端。
  * 支援「單筆刪除」與「一鍵清空」歷史紀錄。
  * 提供「下載 CSV」功能，方便將紀錄匯出至 Excel 長期追蹤。
* **🔍 運動友善介面**：專為運動情境設計的大字體，搭配直覺的背景顏色變化（快走為淺橘色，慢跑為淺綠色），即使手機放得稍遠也能一眼看清。

## 🚀 如何使用與安裝

本程式已透過 GitHub Pages 部署，您可以直接點擊專屬網址開啟使用。

### 📲 將它變成手機主畫面 App (推薦)
為了獲得最佳體驗，建議將此網頁加入手機主畫面，它將會像一個真正的 App 一樣運作（全螢幕且無網址列）：

**🍎 iOS (iPhone/iPad) 用戶：**
1. 務必使用內建的 **Safari 瀏覽器** 開啟本網頁連結。
2. 點擊螢幕下方中央的 **「分享」圖示**（一個正方形加上向上的箭頭）。
3. 在選單中往下滑，選擇 **「加入主畫面」** (Add to Home Screen)。
4. 點擊右上角的「新增」，回到桌面即可看到專屬的 App 圖示！

**🤖 Android 用戶：**
1. 使用 **Chrome 瀏覽器** 開啟本網頁連結。
2. 點擊右上角的 **「三個點」** 選單。
3. 選擇 **「加到主畫面」**，確認新增即可。

## 🛠️ 技術說明
* 本專案僅使用純 HTML, CSS (大字體/響應式設計) 與 Vanilla JavaScript 開發，無依賴任何外部框架。
* 使用 `LocalStorage` 進行本地資料儲存，保護使用者隱私，紀錄不會上傳至任何外部伺服器。
* 使用 `Screen Wake Lock API` 實現防休眠功能（需在 HTTPS 環境下執行）。
* 使用 `Web Audio API` 生成提示音效。
